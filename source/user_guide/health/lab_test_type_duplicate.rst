=============================
:bi:`Lab Test Type Duplicate`
=============================

    #. Executar a Ação :bi:`Lab Test Type Duplicate` para o *Lab Test Type* aberto:

        #. Acessar a *View* *Lab Test Types*:

            * Menu de acesso:
                * :bi:`Health` » :bi:`Configuration` » :bi:`Lab Test Types`

        #. Abrir o Formulário do *Lab Test Type* desejado.

        #. Executar a Ação ":bi:`Lab Test Type Duplicate`":

            * Parâmetros apresentados:
                * *New Lab Test Type*
                * *New Lab Test Type Code*

            #. Utilize o botão :bi:`Duplicate` para executar a Ação.

            #. Utilize o botão :bi:`Cancel` para cancelar a execução da Ação.
