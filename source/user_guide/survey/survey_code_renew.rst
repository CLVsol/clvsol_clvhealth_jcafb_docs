=======================
:bi:`Survey Code Renew`
=======================

    #. Executar a Ação :bi:`Survey Code Renew` para a Pesquisa aberta:

        #. Acessar a *View* *Pesquisas*:

            * Menu de acesso:
                * **Pesquisas** » **Pesquisas**

        #. Abrir o Formulário da Pesquisa desejada.

        #. Exercutar a Ação ":bi:`Survey Code Renew`":

            * Parâmetros apresentados:
                * Nenhum

            #. Utilize o botão :bi:`Code Renew` para executar a Ação.

            #. Utilize o botão :bi:`Cancel` para cancelar a execução da Ação.
