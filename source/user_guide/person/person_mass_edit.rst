=====================================
Atualizar dados em massa de Pessoa(s)
=====================================

    #. [tkl-odoo12-jcafb-vm] Executar a Ação *Person Mass Edit* de Pessoa(s):

        #. Acessar a *View* *Persons*:

            * Menu de acesso:
                * *Community* > *Community* > *Persons*

        #. Selecionar a(s) Pessoa(s) desejada(s).

        #. Exercutar a Ação "**Person Mass Edit**":

            * Parâmetros apresentados:
                * *Global Tags*
                * *Categories:*
                * *Markers*

            #. Utilize o botão *Mass Edit* para executar a Ação.

            #. Utilize o botão *Cancel* para cancelar a execução da Ação.
