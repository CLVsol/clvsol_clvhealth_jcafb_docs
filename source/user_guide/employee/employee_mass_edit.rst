========================
:bi:`Employee Mass Edit`
========================

    #. Executar a ação :bi:`Employee Mass Edit` de Funcionário(s):

        #. Acessar a *view* :bi:`Employees`:

            * Menu de acesso:
                * **Funcionários** » :bi:`Employees` » :bi:`Employees`

        #. Selecionar o(s) Funcionário(s) desejado(s).

        #. Exercutar a ação "**Employee Mass Edit**":

            * Parâmetros apresentados:
                * *Job*
                * *Department*
                * *Global Tags*
                * *Phase*

            #. Utilize o botão :bi:`Mass Edit` para executar a ação.

            #. Utilize o botão :bi:`Cancel` para cancelar a execução da ação.
