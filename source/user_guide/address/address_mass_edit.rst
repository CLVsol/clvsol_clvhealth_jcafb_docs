=======================================
Atualizar dados em massa de Endereço(s)
=======================================

    #. [tkl-odoo12-jcafb-vm] Executar a Ação *Address Mass Edit* de Endereço(s):

        #. Acessar a *View* *Addresss*:

            * Menu de acesso:
                * *Community* > *Community* > *Addresss*

        #. Selecionar a(s) Endereço(s) desejada(s).

        #. Exercutar a Ação "**Address Mass Edit**":

            * Parâmetros apresentados:
                * *Global Tags*
                * *Categories:*
                * *Markers*

            #. Utilize o botão *Mass Edit* para executar a Ação.

            #. Utilize o botão *Cancel* para cancelar a execução da Ação.
