.. index:: Document Mass Edit

========================
:bi:`Document Mass Edit`
========================

    #. [tkl-odoo12-jcafb-vm] Executar a Ação *Document Mass Edit* de Documento(s):

        #. Acessar a *View* *Documents*:

            * Menu de acesso:
                * *Community* > *Community* > *Documents*

        #. Selecionar a(s) Documento(s) desejada(s).

        #. Executar a Ação "**Document Mass Edit**":

            * Parâmetros apresentados:
                * *Global Tags*
                * *Categories*
                * *Documnent Type*
                * *Base Document*
                * *Document Date*
                * *Foreseen Date*
                * *Deadline*
                * *Phase*
                * *Refers to*

            #. Utilize o botão *Mass Edit* para executar a Ação.

            #. Utilize o botão *Cancel* para cancelar a execução da Ação.
