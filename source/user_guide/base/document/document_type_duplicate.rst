.. index:: Document Type Duplicate

=============================
:bi:`Document Type Duplicate`
=============================

    #. Executar a Ação :bi:`Document Type Duplicate` para o *Document Type* aberto:

        #. Acessar a *View* *Document Types*:

            * Menu de acesso:
                * **Base** » **Configuration** » **Document** » **Types**

        #. Abrir o Formulário do *Document Type* desejado.

        #. Exercutar a Ação ":bi:`Document Type Duplicate`":

            * Parâmetros apresentados:
                * *New Document Type*
                * *New Document Type Code*
                * *New Document Type Notes*

            #. Utilize o botão :bi:`Duplicate` para executar a Ação.

            #. Utilize o botão :bi:`Cancel` para cancelar a execução da Ação.
