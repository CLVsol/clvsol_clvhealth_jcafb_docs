=============================
:bi:`Person Associate to Set`
=============================

    #. Executar a Ação :bi:`Person Associate to Set` para as Pessoas selecionadas:

        #. Acessar a *View* *Persons*:

            * Menu de acesso:
                * :bi:`Community` » :bi:`Community` » :bi:`Persons`

        #. Selecionar a(s) Pessoa(s) desejada(s).

        #. Exercutar a Ação ":bi:`Person Associate to Set`":

            * Parâmetros apresentados:
                * *Create new Set*
                * *Set*, se *Create new Set* estiver **desmarcado**
                * *Set Name*, se *Create new Set* estiver **marcado**

            #. Utilize o botão :bi:`Associate to Set` para executar a Ação.

            #. Utilize o botão :bi:`Cancel` para cancelar a execução da Ação.
