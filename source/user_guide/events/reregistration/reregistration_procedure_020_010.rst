==================================================================
Criação do **Cadastro Auxiliar** para uma **Pessoa já cadastrada**
==================================================================

    #. Executar o procedimento ":doc:`/user_guide/community/person/person_associate_to_person_aux`" para o registro :bi:`Person` aberto durante a execução do procedimento anterior (":doc:`reregistration_procedure_010_010`" ou ":doc:`reregistration_procedure_010_020`"):

        #. Exercutar a Ação ":bi:`Person Associate to Person (Aux)`":

            #. Parâmetros apresentados:
                * *Create new Person (Aux)*
                * *Create new Family (Aux)*
                * *Create new Address (Aux)*

            #. Utilize o botão **[** :bi:`Associate do Perton (Aux)` **]**  para executar a Ação.

        #. Utilizar o registro da Pessoa apresentado na *view* :bi:`Persons (Aux)`.
