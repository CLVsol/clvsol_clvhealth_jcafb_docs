=======================================
Procura por uma Família em **Contatos**
=======================================

    Este é o método de pesquisa mais abrangente, e o preferencial quando não se dispuser do Código da Família a ser pesquisada.

    A procura por uma Família utilizando a *view* **Contatos** é realizada quando se dispõe somente da informação do **Nome** da mesma (situação em que é incerta a existência de um cadastro para essa Família).

    Neste caso não é possível realizar a pesquisa utilizando o Codigo da Família.
